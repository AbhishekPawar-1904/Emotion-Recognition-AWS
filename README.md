# Emotion Recognition using AWS

This project detects facial emotions using Amazon Rekognition.

## AWS Services Used

- Amazon S3
- Amazon Rekognition
- AWS Lambda
- API Gateway
- Amazon SQS
- Amazon DynamoDB

## Features

- Capture image from webcam
- Upload image to S3
- Detect emotions using Amazon Rekognition
- Store results in DynamoDB
- Display results on the frontend
